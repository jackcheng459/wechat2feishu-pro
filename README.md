# 🚀 WeChat2Feishu-Pro (适配 OpenClaw)
### 微信文章一键转存飞书：高保真、智能路由、无人值守。

**WeChat2Feishu-Pro** 是一个专门为飞书用户打造的文章剪藏系统。它完美解决了微信公众号文章转存至飞书时最棘手的：图片防盗链、排版错乱、以及 Wiki 知识库权限等痛点。配合 **OpenClaw** 插件，您可以获得像私人助理一样的转存体验。

---

## ✨ 为什么选择 Pro 版本？

- **🛡️ 捕获二进制原图**：利用 Playwright 响应拦截技术，直接抓取原图二进制数据，彻底杜绝微信 CDN 防盗链导致的图片损坏。
- **📐 比例精准适配**：自动识别图片宽高并同步注入飞书，拒绝图片拉伸变形。
- **🤖 智能存储路由**：
    - **优先级 1**：即时指定目录。
    - **优先级 2**：记住您的默认保存偏好。
    - **优先级 3**：自动兜底存入飞书云文档主页。
- **🕒 无人值守自动备份**：发出询问 5 分钟未响应，机器人将按照智能路由自动完成转存，绝不丢失文章。
- **📁 双轨存档**：云端存入飞书，本地同步生成 `README.md` + `images/` 原图文件夹，实现个人知识库的永久本地化。

## 🛠️ 快速安装 (小白友好)

### 1. 准备飞书“通行证”
1.  登录 [飞书开放平台](https://open.feishu.cn/app) 创建一个自建应用。
2.  记录 **App ID** 和 **App Secret**。
3.  **开通权限**：搜索并开启“查看、编辑、管理云空间、云文档、知识库”以及“发送消息”的所有必要权限。

### 2. 一键环境部署
在终端（Terminal）中运行：
```bash
bash setup.sh
```

### 3. 配置与首次授权
1.  在 `.env` 文件中填入 ID 和 Secret。
2.  运行：`source .venv/bin/activate && python auth.py login` 确认授权。
3.  **权限分配 (关键)**：将您的自建应用（机器人）添加为目标文件夹或知识库空间的**管理员/协作者**（点击飞书文档右上角的“分享”按钮即可添加）。

---

## 🤖 OpenClaw 技能集成
1.  进入项目 `skills/` 目录。
2.  将 `wechat2feishu-pro.md` 拷贝到您的 OpenClaw 技能库。
3.  **开启静默模式**：在 `.env` 中设置 `USE_TENANT_MODE=true`，实现全自动后台运行，不再有网页弹窗。

---

## 📂 项目结构
- `main.py`: CLI 核心入口。
- `feishu.py`: 飞书 API 深度集成模块（支持 Wiki 挂载）。
- `scraper.py`: Playwright 高级采集引擎。
- `processor.py`: “手术刀级” Markdown 格式清洗器。
- `exports/`: 本地图文备份存放地。
- `docs/`: 包含技术架构图和开发沉淀日志。
