# 🚀 WeChat2Feishu (适配 OpenClaw)
### 微信文章一键转存飞书，从此告别图片损坏和排版错乱！

WeChat2Feishu 是一款专门为**非技术用户**打造的自动化工具。它可以帮你把喜欢的微信公众号文章，一键完美转存到你的飞书云文档或知识库（Wiki）中，甚至在本地自动备份一份。

---

## 📱 典型使用场景
**把文章链接转发给飞书机器人** → **机器人自动抓取** → **你选个文件夹** → **秒级完成转存！**

---

## 🛠️ 三步搞定安装（小白级教程）

### 第一步：准备飞书“通行证”
1.  登录 [飞书开放平台](https://open.feishu.cn/app) 创建一个“自建应用”。
2.  在“凭证与基础信息”里找到 **App ID** 和 **App Secret**（先记下来）。
3.  在“权限管理”里搜索并开启：**云文档、云空间、知识库、消息** 相关的权限。

### 第二步：一键安装程序
打开你的终端（Mac 叫 Terminal），进入项目目录后，直接粘贴运行下面这行命令：
```bash
bash setup.sh
```
*这会自动帮你装好所有需要的东西，你只需要静候 1-2 分钟。*

### 第三步：配置与授权
1.  打开项目里的 `.env` 文件，把第一步记下来的 ID 和 Secret 填进去。
2.  在终端运行：
    ```bash
    source .venv/bin/activate
    python auth.py login
    ```
3.  在弹出的网页里点一下“确认授权”。**大功告成！**

## 🤖 给 OpenClaw 用户的特别福利
如果你正在使用 OpenClaw 插件：
1.  找到本项目中的 `skills/wechat2feishu.md` 文件。
2.  把它拷贝到你的 OpenClaw 技能存放目录。
3.  从此，你只需要在飞书里给机器人发个链接，剩下的它全包了！

---

## ✨ 为什么选它？
- **原图保存**：彻底解决微信图片“防盗链”导致的图片无法显示问题。
- **比例完美**：图片不会被拉伸或压缩，100% 还原原文。
- **排版高保真**：代码块、标题、引用格式统统保留。
- **本地+云端双备份**：即使原文被删，你的飞书和硬盘里也永远有一份。
